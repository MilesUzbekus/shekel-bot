# shekel-bot — Telegram-бот арбитража шекель/рубль

Личный бот для учёта обменных сделок ILS↔RUB с курсом, аналитикой и прогнозом.

## Возможности
- Запись сделок в Google-таблицу.
- Курс ILS/RUB напрямую с Google Finance.
- Прогноз курса (доверительный конус + bias).
- Аналитика по сделкам и P&L.
- Самовосстановление и авто-переключение direct↔proxy для доступа к Telegram API.

## Стек
Python · python-telegram-bot · Google Sheets API · LLM-прогноз · портативная упаковка.

## Структура
- `src/` — модули: бот, сделки/леджер, аналитика, прогноз, новости, источники курса
- `demo.py` — демонстрационный запуск
- `.env.example` — конфигурация без секретов

Секреты (токен бота, ключи API, Google-креды) берутся из `.env`/окружения.
В репозиторий вынесен только исходный код (без runtime, секретов и данных).
Статус: в продакшене (приватный бот).
